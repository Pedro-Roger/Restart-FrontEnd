# Restart Financial Evolution App

Expo/React Native prototype generated from the Google Stitch project `7721913470452622854`.

## Run

```bash
npm install
npm run start
```

The current static web export is in `dist/` and can be previewed at `http://localhost:3000` while the local preview server is running.

## Stitch Assets

- Screenshots: `assets/stitch/screens/`
- Original Stitch HTML exports: `stitch-code/`
- Manifest: `stitch-code/manifest.json`

To fetch the Stitch assets again:

```bash
STITCH_API_KEY="your-key" python3 scripts/fetch_stitch_assets.py
```
